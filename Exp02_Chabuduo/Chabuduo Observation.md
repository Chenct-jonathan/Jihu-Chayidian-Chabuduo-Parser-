- conjunction + ... + 差不多

  - 和諾可差不多

- 差不多 + (verbP)? + time

  - 差不多有十年

- 也 + (應該)? + 差不多

  - 情況也差不多

- [Quantifier 也都] + 差不多 

  - 情況都差不多 

- verb 起來 + 差不多

  - 聽起來差不多嗎

  - 各縣市情況差不多 

- 任意字詞 + 則 + 差不多

  - 基本設計則差不多

- entity + range + 差不多

  - 好像意思上差不多 

- verb + 得 + 差不多 + (aspect 了)

  - 海青病得差不多了

- entity + (已經)? + 差不多 + 了

  - 生命已經差不多了

- 還 + 差不多

  - 說像海盜還差不多

- 差不多 + ... + 時/的時候

  - 差不多到一半路程時

- 差不多 + (是)? + num

  - 差不多是二十八歲吧

- 差不多 + 都 + ([已經])? + verb + (了|modifier) * (telicity)

  - 差不多都已查證清楚

- verb + (的)? + entity + 差不多  * 

  - 所使用的工具差不多

  - 那我不就差不多了嗎

- 差不多 + Modifier 

  - 雖然看起來差不多亮

  - 夜晚差不多都在村子中

  - 身上器官也壞得差不多
